Buddy AI Calling Agent

Overview
This repo contains a full-stack AI calling agent with a FastAPI backend and a Vite + React frontend.

Structure
- backend: FastAPI service, routes, and integrations
- frontend: React UI built with Vite

Quick start
Backend
1) Open a terminal in the backend folder.
2) Install deps: pip install -r requirements.txt
3) Run the API: python -m uvicorn app.main:app --reload

Frontend
1) Open a terminal in the frontend folder.
2) Install deps: npm install
3) Run the app: npm run dev

Notes
- Create a backend .env with any required credentials for your environment.
